# YOLO-MSG

PV Defect Detection Dataset

This dataset is used in the paper:
“YOLO-MSG: A Lightweight and Real-Time Photovoltaic Defect Detection Algorithm for Edge Computing.”

It contains 6,483 images, divided into training, validation, and test sets with a ratio of 7:2:1, covering three categories: cracked, dusty, and panel.

The dataset is derived from the Roboflow project:
https://universe.roboflow.com/susan-ifblr/panel-solar-bw945

To improve dataset quality, we performed the following refinements:

Removed underrepresented categories

Filtered out images with poor acquisition quality

Corrected or removed inaccurate annotations
