# YOLO-MSG

**PV Defect Detection Dataset**

This dataset is used in the paper:
**“YOLO-MSG: A Lightweight and Real-Time Photovoltaic Defect Detection Algorithm for Edge Computing.”**

**Dataset Details**

Total images: 6,483

Split ratio: 70% training / 20% validation / 10% test

Classes: cracked, dusty, panel

**Source**

The dataset is derived from Roboflow:
https://universe.roboflow.com/susan-ifblr/panel-solar-bw945

**Modifications**

Removed underrepresented categories

Filtered out low-quality images and inaccurate annotations
